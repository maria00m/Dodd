# Davood Support Page

A beautiful, responsive web page created to support my friend Davood after his breakup. This page reminds him that he deserves everything good in life.

## Features
- Fully responsive design (mobile, tablet, desktop)
- Personalized messages for Davood
- Interactive elements (clickable buttons, confetti)
- Stylish animations and gradients
- Heartwarming content

## Deployment
This site is deployed on Vercel as a static website.

## How to Use
1. Open the deployed URL
2. Read the personalized message
3. Click buttons for encouragement
4. Share with Davood when he needs support
